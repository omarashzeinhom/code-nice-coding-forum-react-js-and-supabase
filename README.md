# StackOverFlow Clone in Next.js and Neo4j


## Start By
```yarn dev```


### Packages Installed 
1. [neo4j-driver]()
 
```yarn add neo4j-driver```

1. [nextAuth.js Neo4j getting Started](https://next-auth.js.org/adapters/neo4j)

```yarn add next-auth @next-auth/neo4j-adapter neo4j-driver```
